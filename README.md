# 🔢 Number Guesser Game

## 📌 Overview
This is an interactive **Number Guesser Game** built using HTML, CSS, and JavaScript. The player competes against the computer by guessing a number between 0 and 9, with scores tracked across multiple rounds.

---

## 🎮 How the Game Works
- A random target number is generated each round
- Both the **player** and the **computer** make a guess
- The closest guess to the target number wins the round
- Scores are updated after each round
- The game continues with a “Next Round” button

---

## ⚙️ Technologies Used
- HTML5
- CSS3
- JavaScript

---

## 🧱 Features
- Player vs Computer gameplay
- Score tracking system
- Multiple rounds
- Interactive input controls (+ / - buttons)
- Input validation (0–9 range)
- Dynamic UI updates using JavaScript
